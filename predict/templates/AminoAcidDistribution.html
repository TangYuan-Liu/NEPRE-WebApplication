Amino Acid Pairs Distribution

Pair Distribution Preference

Each kind amino acid has own ditribution preference. We counted the existing PDB dataset (Before June 2017) and analyzed the distribution of each kind of amino acid pairs.

For each kind of amino acid pair, we establish a spherical coordinate system using r,θ,φ. For both Nepre-R and Nepre-F, int the direction of θ,φ, each of them are divided into 20 parts. In the direction r, only one layer is included.

For protein structure assessment, our experiment shows using one layer with cutoff 6 angstrom gets the best recognition. But we also find some interesting thing in other layers of amino acid pairs. The following 4 figures show pairs distribution preference of CYS-GLY in different layers.

The following two table shows the detail of layers subregion of the coordinate system.

It shows that each kind of amino acid pair in different layer has some specific concentrated distribution area, which suggests that amino acid pairs tend to appear in these regions.

Application

Each amino acid pair has a center amino acid and a surround amino acid(CYS-GLY is different from GLY-CYS). Choose a pair and select the layer. Then the following application will return the distribution preference of that kind of pair.
