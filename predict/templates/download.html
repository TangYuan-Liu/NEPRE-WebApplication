Offline System

Online system only provide basic function for protein structure prediction. So we established an offline software for users to complete personalized tasks. The offline system is writeen by using Python2.7 and GUI is build by using PYQT4. The Nepre.exe has included all packages it need.

Extended functions are as follows,

1. Users can use their own PDB dataset to get statistical distribution of amino acid and generate energy matrix.

2. Offline system provide some analyzing tools for users to get detailed information.

3. Offline system can provide a stable calculation tool for large workload.
